# Archive Index

This directory holds historical material that is **not** part of the
authoritative project state. Everything here is retained for traceability
only.

**Do not load or scan the contents of `archive/` unless a task explicitly
requires it.** None of these files describe the current build, current test
results, or current readiness verdict — see `docs/CURRENT_BUILD_STATE.md`,
`reports/`, and `PROJECT_MANIFEST.md` for the current state.

## Contents

- `brainstorms/` - early exploratory notes and design brainstorms, superseded
  by the implemented workflows and `docs/ARCHITECTURE.md`.
- `prompts-history/` - prior session prompts/instructions, retained for
  traceability of how the project evolved.
- `handoffs/` - superseded session handoff documents
  (e.g. `SESSION_HANDOFF_PHASE_2_SUPERSEDED.md`).
- `scripts-history/` - one-off or superseded maintenance scripts
  (e.g. `update-phase4b-validator-fix.ps1`), kept for reference only and not
  intended to be re-run.
